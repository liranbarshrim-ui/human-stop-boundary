Decision Accountability Record (DAR)

Matrix Audit is an independent governance framework focused on accountability in autonomous systems.

Decision Accountability Record (DAR) is a governance principle defining named human stop authority in autonomous systems.

PRINCIPLE

The Authority to Initiate is not the Authority to Terminate.

Automated systems may execute decisions. Accountability remains human.

FORMAL DEFINITION

Decision Accountability Record (DAR) — A governance principle assigning named human authority responsible for stopping automated systems before irreversible execution occurs.

DAR defines the governance principle that assigns a named human authority responsible for stopping automated systems before irreversible execution occurs.

PURPOSE

DAR ensures that automated systems operating in critical environments cannot execute irreversible actions without an identified human authority responsible for halting execution.

REFERENCE CASE

The 2012 Knight Capital event demonstrated how automated systems can execute catastrophic actions when no clearly defined human authority exists to stop execution.

KEY QUESTION

Who holds the authority to stop the system when execution becomes irreversible?

BY NAME

KEY CONCEPTS

Human Stop Authority — the explicit mandate granted to a named individual to halt automated execution when necessary.

Irreversible Execution — automated actions that cannot be safely reversed once initiated.

Governance Boundary — the structural point where automated execution must yield to human authority.

DAR STRUCTURE

1. System under governance
2. Irreversible execution boundary
3. Named human stop authority
4. Conditions under which stop authority may be exercised

What is DAR?
