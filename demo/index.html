Question 01
When does this decision become irreversible?

The point after which the action cannot be undone. Money moved. Trade executed. Record written.

Required.
Question 02
Who, by name, can stop it before that moment?

Not a role. Not a team. One person — first name, last name — with a prior signed commitment to act.

Please enter a name or explicitly write "No one defined."
Question 03
Does that person have a real mechanism to stop it?

A button, override, API call, signed authority. Not a meeting. Not an escalation path.
